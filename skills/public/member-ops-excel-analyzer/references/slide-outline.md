# 页面规划

## 目标

将固定模板 Excel 自动转成一套适合后续 PPT 制作的标准页面内容。

要求：

- 页面数量固定
- 信息密度适中
- 每页都具备汇报感
- PPT skill 无需再次理解复杂业务逻辑
- 不依赖任何历史 PPT 才能成立

## 标准页面清单

### 1. 封面

- 标题
- 副标题
- 数据来源说明

### 2. 分析框架

- 分析范围
- 可计算主题
- 标签与行为维度

### 3. 会员概况

- 指标卡：会员总数、身份标注、性别标注、平均年龄
- 图表：年龄分布
- 图表：身份 Top 结构
- 图表：会员等级分布
- 洞察

### 4. 一物一码来源已挂靠会员

- 指标卡：可控门店数、挂靠人数、挂靠占比、抽奖参与
- 主表格：门店排名

### 5. 一物一码待挂靠会员

- 指标卡：待挂靠人数、核心档位占比、重点区域
- 表格：按工种扫码档位
- 表格：按区域待挂靠人数

### 6. 会员消费

- 指标卡：进店消费、扫码消费、双行为交集、去重活跃
- 表格：门店类型消费表现
- 图表：最近一次消费月份
- 表格：等级消费贡献
- 风险卡：流失风险、需关注、SVIP

### 7. 上门基检

- 指标卡：覆盖会员、基检次数、券核销、连带消费
- 表格：重点门店基检表现
- 页面解读

### 8. 会员日

- 指标卡：参与人数、参与率、人均参与次数、SVIP参与率
- 图表：月度参与趋势
- 图表：按店型参与率
- 表格：次数分布
- 表格：等级参与与转化

### 9. 会员分层与风险池

- 指标卡：高活跃核心、消费待维系、待转化潜力、沉默会员
- 表格：风险池定义
- 页面建议

### 10. 门店经营机会优先级

- 指标卡：最高机会分、优先门店、低消费转化样本
- 图表：机会分排序
- 表格：门店标签

### 11. 转化路径与运营建议

- 图表：关键触点覆盖
- 指标卡：优先运营池、风险池、优先门店、高价值样板
- 建议话术

## slide_spec 建议字段

每页输出结构建议：

```json
{
  "slide_id": "member_overview",
  "title": "会员概况：规模、标签覆盖与结构分布",
  "subtitle": "口径说明……",
  "layout": "kpi_plus_two_charts",
  "kpis": [],
  "charts": [],
  "tables": [],
  "insights": [],
  "notes": {
    "source_sheets": [],
    "metric_logic": ""
  }
}
```

## 与 PPT skill 的协作方式

建议 PPT skill 只负责：

- 版式
- 样式
- 图表绘制
- 表格视觉处理
- 导出 `.pptx`

建议当前 skill 负责：

- 标题
- 页面模块
- 数据内容
- 洞察文案
- 口径说明
